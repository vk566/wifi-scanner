\# WiFi RF Scanner 🚀



An WiFi signal scanning and tracking system.



\## 🔥 Features

\- Real-time WiFi scanning

\- RSSI signal classification using Deep Learning

\- Signal direction tracking

\- MAC address detection

\- Live graph visualization



\## 🧠 Tech Stack

\- Python (Flask)

\- TensorFlow / Keras

\- PyWiFi

\- Chart.js



\## ▶️ How to Run



1\. Train models: python train\_model.py


2\. Run app: python app.py



\## 📊 Output

\- WiFi networks list

\- Signal strength classification

\- Direction tracking

\- Live RSSI graph



\## 🚀 Future Scope

\- RF signal integration

\- Threat detection
